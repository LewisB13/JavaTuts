# Java User Input Tasks ☕

## Task 1 — Import Scanner
Add the Scanner import to your Java program.

```java

```

---

## Task 2 — Create a Scanner
Create a Scanner object called `input`.

```java
Scanner input = new Scanner(System.in);
```

---

## Task 3 — Ask For A Name


## Task 4 — Store User Input
Create a String variable that stores the user's name.


## Task 5 — Display The Name
